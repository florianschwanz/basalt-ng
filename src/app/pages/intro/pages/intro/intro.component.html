This is Basalt
Basalt is a tool that helps you
  • keep track of work you have done
  • plan ahead
  • manage contacts
Define tasks
Tasks are the things you want to accomplish and allows you to define
  • a description
  • a priority
  • a due date
  • an associated project
  • a definition of done
Make use of tasklets
A tasklet is a piece of work which can be associated with a task. Tasklets can be of different types such as
  • simple action
  • call
  • meeting
  • checklist
All your tasklets will be displayed in a timeline.
Store your data securely
Basalt makes use of IndexedDB - a database which is part of your browser. No data is transferred to the cloud. Everything stays on your device.
How do you work?
Select the aspects you want to use (you can change this later)
Start the app and create your first tasklet. Please click the bullhorn icon if you want to provide any feedback regarding the app.
